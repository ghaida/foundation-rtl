foundation-rtl by ghaida

Localized version of Foundation for use with right-to-left languages. Foundation is an easy to use, powerful, and flexible framework for building prototypes and production code on any kind of device. Start here, build everywhere. هذه النسخة المعدلة من الأساس للاستخدام بالعربية أو بأي لغة تقرأ من اليمين للشمال.
